# Morpion 2.0

## Description

Morpion 2.0 est une version améliorée du jeu classique de morpion (Tic-Tac-Toe). Ce jeu permet aux joueurs de personnaliser leurs symboles en choisissant entre des formes traditionnelles (X et O) ou des images de Pokémon pour une expérience plus fun et engageante.

## Fonctionnalités

- **Choix de skins personnalisés** : Les joueurs peuvent sélectionner des symboles classiques (X/O) ou des images de Pokémon pour représenter leurs jetons.
- **Aperçu en temps réel** : Un aperçu des skins choisis est affiché avant le début de la partie.
- **Interface utilisateur intuitive** : Design moderne et responsive avec un panneau de contrôle et une grille de jeu claire.
- **Détection automatique des victoires** : Le jeu détecte automatiquement les combinaisons gagnantes et les matchs nuls.
- **Boutons de contrôle** : Démarrer une nouvelle partie ou rejouer après la fin d'une partie.

## Comment jouer

1. Ouvrez le fichier `index.html` dans votre navigateur web préféré.
2. Choisissez les skins pour le Joueur 1 et le Joueur 2 dans les menus déroulants.
3. Cliquez sur le bouton "Démarrer" pour lancer la partie.
4. Les joueurs alternent les tours en cliquant sur les cases vides de la grille.
5. Le premier joueur à aligner trois symboles identiques (horizontalement, verticalement ou en diagonale) gagne.
6. En cas de match nul, la grille est remplie sans gagnant.
7. Utilisez le bouton "Rejouer" pour commencer une nouvelle partie.

## Technologies utilisées

- **HTML5** : Structure de la page web.
- **CSS3** : Styles et mise en page responsive.
- **JavaScript (ES6+)** : Logique du jeu et interactions utilisateur.
